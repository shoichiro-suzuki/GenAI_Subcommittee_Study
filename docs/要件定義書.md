# Todo管理アプリ - 要件定義書

## 1. プロジェクト概要

### 1.1 アプリケーション名
**SimpleTodo**

### 1.2 目的
個人用の軽量Todo管理アプリ。実行ファイル単体で動作し、シンプルな操作でタスク管理を実現。

### 1.3 対象ユーザー
- 単一ユーザー（個人利用）
- 簡易GUIベースのシンプルな操作を好むユーザー

## 2. 技術仕様

### 2.1 技術スタック
- **言語**: Python 3.8+
- **GUIフレームワーク**: tkinter（標準ライブラリ）
- **配布形式**: PyInstaller による単一EXEファイル
- **データ保存**: JSON形式（EXEと同ディレクトリ）

### 2.2 動作環境
- **OS**: Windows
- **必要ファイル**:
  - `SimpleTodo.exe`（実行ファイル）
  - `todos.json`（データファイル、自動生成）

## 3. 機能要件

### 3.1 基本機能
| 機能 | 説明 |
|-----|------|
| Todo追加 | 新しいタスクを追加 |
| Todo一覧表示 | 全タスクの表示（ID、タイトル、状態） |
| 状態切り替え | 完了/未完了の切り替え |
| Todo削除 | 指定タスクの削除 |

### 3.2 データ操作
- **自動保存**: 変更時に即座にJSON保存
- **自動読み込み**: 起動時にJSONから自動読み込み
- **初回起動**: JSONファイルが存在しない場合は自動生成

## 4. 非機能要件

### 4.1 パフォーマンス
- 起動時間: 1秒以内
- 操作レスポンス: 即座（体感遅延なし）

### 4.2 可用性
- EXE単体で動作（外部依存なし）
- JSONファイル破損時の安全な処理

### 4.3 保守性
- シンプルなコード構造
- 機能拡張が容易な設計

## 5. 制約事項

### 5.1 技術的制約
- ネットワーク機能なし
- マルチユーザー非対応
- データ同期機能なし

### 5.2 運用制約
- データバックアップはユーザー責任
- JSONファイルの直接編集は非推奨

## 6. 将来拡張の可能性
- 優先度設定
- 期限管理
- カテゴリ/タグ機能
- 検索機能
- データエクスポート機能
